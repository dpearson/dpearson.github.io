Ending support for node-identi.ca

A little while ago, the team behind identi.ca announced that they were pulling the plug on the site and moving everything over to pump.io. I'm sure there will be a ton of benefits for identi.ca's users.

The fact is, I'm not one of them. I never really was. I flirted with bringing NFLScoreBot over to identi.ca, which was why I built node-identi.ca to begin with. It was completely the wrong audience, but I had really only done it for fun.

Pump.io will have a very different API, rather than the simple, Twitter-compatible API identi.ca exposed. Therefore, I won't be updating this library to support it; the effort just isn't worth it. I'll leave the code live, as the authorization tool is a decent example of an OAuth flow for node.js.


If you like this post, you might want to check out my current project, NHLScoreBot.
