NHLScoreBot, Year II

TL;DR: Hockey's back. Follow NHLScoreBot in English or French for NHL scores and game summaries on Twitter.

With the NHL lockout finally resolved, it's time for another year of hockey. There's a few new features:

  1. Locations. Wanted to know who the home team was? Now, your Twitter client will show a nice map on the summary's detail view.

  2. French support. This feature is something I added last year to NFLScoreBot, and the backend update to NHLScoreBot brought it into the fold.

I've also open sourced some more code. The arena location data I'm using is available on Github (and you can grab it with a standard HTTP request).

My code for using the identi.ca API with node.js is also on Github and NPM.


If you like this post, you might want to check out my current project, NHLScoreBot.
